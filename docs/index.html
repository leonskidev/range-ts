Rust-like ranges in Deno (TypeScript)!

I love Rust's syntax, and so I thought it would be fun to transfer it, in essance at least, over to Deno! It's not done yet, I'm testing out some ideas to make it even more awesome at the moment.

Before you decide to use this in a production environment, don't. This is really just a fancy fun wrapper around the built-in Array.prototype.slice method.

All the code you see here can and should be substituted for that in a production environment. I'm not here to create another silly dependency that can, and probably will, break!

Docs + Examples

Everything

The simplest syntax gets a copy of all the items in the array:

          range("..")(arr); // [1, 2, "hello", "world"]

You can imagine this as saying, get items from index 0 to index arr.length - 1.

From

This format gets a copy of all the items after an index:

          range("2..")(arr); // ["hello", "world"]

You can imagine this as saying, get items from index 2 to index arr.length - 1.

To

This time we want to get all the items up to an index. But there are two common ways to do this, exclusively (excluding the item at index), or inclusively (including the item at index).


To exclusively get items up to an index, we would do this:

          range("..3")(arr); // [1, 2]

You can imagine this as saying, get items from index 0 to index 2.


And to inclusively get items up to an index, we would do this:

          range("..=3")(arr); // [1, 2, "hello"]

You can imagine this as saying, get items from index 0 to index 2.

Within

Just like with the above situation, we can either get items exclusively: 

          range("1..3")(arr); // [2, "hello"]

You can imagine this as saying, get items from index 1 to index 2.


Or inclusively:

          range("1..=3")(arr); // [2, "hello", "world"]

You can imagine this as saying, get items from index 1 to index 3.
